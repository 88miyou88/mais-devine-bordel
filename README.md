# Mais devine, bordel ! — Prototype V1

Nom affiché : **Mais devine, bordel !**  
Nom technique du dépôt : **mais-devine-bordel**  
Nom du raccourci : **MDB!**

## Contenu de cette V1

- Mode « Deviner les paroles »
- Affichage horizontal
- Compte à rebours 3, 2, 1
- Chronomètre 30, 60, 90 secondes ou durée personnalisée
- Swipe à droite : validé
- Swipe à gauche : passé
- Un simple toucher ne change pas la carte
- Bouton Retour avec correction du score
- Pause
- Résultats détaillés
- Retournement de l’affichage à 180°
- Tentative de maintien de l’écran allumé
- Installation PWA
- Fonctionnement hors ligne après le premier chargement
- Diagnostic copiable

## Installation sur GitHub Pages

Les étapes précises seront réalisées avec Camille, bouton par bouton.

Résumé :
1. Créer un dépôt public nommé `mais-devine-bordel`.
2. Envoyer tous les fichiers de ce dossier à la racine du dépôt.
3. Ouvrir `Settings` → `Pages`.
4. Choisir `Deploy from a branch`.
5. Choisir la branche `main` et le dossier `/root`.
6. Ouvrir l’adresse publiée.
7. Dans Chrome, choisir `Ajouter à l’écran d’accueil` ou `Installer l’application`.

## Test local facultatif

L’application ne doit pas être ouverte directement avec une adresse `file://`.
Elle doit être servie par un serveur web ou GitHub Pages pour que le mode hors ligne fonctionne.
