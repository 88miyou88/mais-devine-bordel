# Mais devine, bordel ! — V0.5.0

Application de jeu installable sur téléphone, pensée pour être utilisée en mode paysage.

## Jouer

1. Sélectionner un ou plusieurs modes.
2. Ouvrir **Configurer** pour choisir les catégories et les difficultés.
3. Choisir la durée, puis lancer la partie.
4. Pendant une manche classique : glisser à droite pour valider, à gauche pour passer.

## Modes

- **Deviner les paroles** : les autres chantent le début ; la personne qui porte le téléphone retrouve la suite.
- **Mimer** : faire deviner uniquement par gestes.
- **Sans le dire !** : faire deviner un mot sans prononcer les mots interdits affichés.
- **Dessine-moi ça !** : révéler la consigne, puis dessiner sur le téléphone ou sur papier. Les cartes faciles valent 1 point, les moyennes 2 et les difficiles 3.

## Dessin sur téléphone

- couleurs à gauche ;
- épaisseur, gomme/crayon, annuler et poubelle à droite ;
- maintenir **Trouvé** ou **Passer** pendant 0,5 seconde ;
- quitter le bouton avant la fin annule immédiatement l’action ;
- la poubelle peut être annulée avec la flèche de retour.

## Personnalisation

**Gérer les cartes** permet d’ajouter, modifier, dupliquer, déplacer, désactiver ou supprimer des cartes et des catégories. Les données personnelles restent enregistrées sur le téléphone. Les paramètres avancés permettent les sauvegardes et les mises à jour des bibliothèques.
