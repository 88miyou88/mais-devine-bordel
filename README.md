# Mais devine, bordel !

**Mais devine, bordel !** est un jeu d’ambiance personnalisable à utiliser sur téléphone en mode paysage.

**Version actuelle : 0.4.2**  
**Nom du raccourci : MDB!**

## Lancer une partie

1. Choisir un ou plusieurs modes de jeu.
2. Toucher **Configurer** pour sélectionner les catégories et les difficultés.
3. Choisir la durée de la manche pour les modes joués sur le front.
4. Appuyer sur **Lancer la partie**.
5. Poser le téléphone horizontalement sur le front, écran tourné vers les autres joueurs.

Pendant les modes classiques :

- glisser vers la **droite** pour valider ;
- glisser vers la **gauche** pour passer ;
- **Retour** annule le dernier swipe et corrige le score ;
- **Pause** arrête temporairement le chrono ;
- **Retourner** pivote l’affichage à 180° ;
- **Terminer** arrête la manche.

Le téléphone vibre différemment selon qu’une carte est validée ou passée. À la fin, le détail des cartes jouées est affiché.

## Deviner les paroles

Les joueurs qui voient l’écran chantent la partie affichée en blanc sur le bon air.

La personne qui tient le téléphone doit retrouver la suite affichée en vert.

Le titre et l’artiste, le film ou la source apparaissent en bas de la carte pour aider les chanteurs à identifier le morceau.

## Mimer

Les joueurs qui voient l’écran doivent faire deviner la consigne uniquement avec des gestes.

Ils ne doivent pas parler, écrire la réponse ou former clairement des lettres avec les doigts.

## Sans le dire !

Les joueurs décrivent le mot ou l’expression affichée afin que la personne tenant le téléphone le devine.

Il est interdit de prononcer :

- la réponse ;
- un mot de la même famille ;
- les mots interdits affichés en rouge.

L’affichage des mots interdits peut être désactivé dans les réglages du mode.

## Dessine-moi ça !

Dans la version actuelle, ce mode se joue seul.

Le joueur choisit :

- le nombre de dessins ;
- les catégories ;
- les difficultés ;
- le temps accordé aux cartes faciles, moyennes et difficiles ;
- l’activation du son à la fin du chrono.

### Déroulement

1. La consigne est d’abord masquée.
2. Le dessinateur appuie sur la carte pour la révéler sans la montrer aux autres.
3. Il choisit :
   - **Je gribouille ici** pour dessiner sur le téléphone ;
   - **Je massacre une feuille** pour dessiner sur papier ;
   - **Passer ce dessin** pour utiliser la tentative sans marquer de point.
4. Le chrono démarre après le choix du support.
5. Pour terminer, il faut maintenir **Trouvé** ou **Passer** pendant une demi-seconde afin d’éviter les clics accidentels.

Il est interdit d’écrire des lettres, des mots ou des chiffres donnant directement la réponse.

### Dessin sur le téléphone

Le canevas comprend :

- plusieurs couleurs sur le côté gauche ;
- un curseur vertical pour l’épaisseur du trait ;
- un bouton alternant entre crayon et gomme ;
- une flèche pour annuler la dernière action ;
- une poubelle rouge pour tout effacer ;
- la possibilité d’annuler un effacement total accidentel ;
- le chrono et le score directement sur la zone de dessin.

### Points

- Facile : **1 point** ;
- Moyen : **2 points** ;
- Difficile : **3 points**.

En cas d’égalité, le temps utilisé peut servir de départage.

## Personnaliser le jeu

Le bouton **Gérer les cartes** permet de :

- ajouter une carte ;
- modifier ou supprimer une carte ;
- activer ou désactiver une carte ;
- changer sa catégorie et sa difficulté ;
- créer, renommer ou supprimer des catégories.

Les modifications personnelles sont enregistrées sur le téléphone utilisé.
